Training Space

The training space helps users to understand various exercises and how to correctly perform them. This will help beginners to understand how to correctly perform a specific exercise while understanding the precise form as well

The Training space provides:

  • Visual representation of different exercises
  • Different muscle groups targeted in a particular exercise
  • Preparation/Pre-requisites
  • Proper Execution & Key Tips

Real-time Form Detection

This feature provides the human touch to the virtual gym experience. Users can get a real-time form analysis on how they are performing a particular exercise

This analysis will be highly beneficial and help users track their progress over time and will be considered as a factor in their leveling up procedure

Users can use their device camera to record their video

This will be a fast process and users will not have to wait for longer times to get their analysis

REP Counter

This feature helps the user count reps in an exercise. The number of accurate reps will be displayed on the screen and their accuracy will be checked based on angles

  • Reps shown on the screen based on form
  • Incomplete reps are not counted thus helping user improve the form
  • Timer is included which can help user time their exercises
  • Analyzing the level of the user by comparing time and accurate reps

Blog Space

Users can create blogs for the community to read. The number of views and the upload date will be available on the blog. The user is allowed to delete or update the blogs and add new ones as well
